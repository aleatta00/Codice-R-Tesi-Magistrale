Repository contenente il codice R utilizzato per la tesi magistrale
“Inferenza bayesiana nei modelli di copule Archimedee:
una parametrizzazione basata sul coefficiente di Kendall e applicazioni ai mercati finanziari”.

Il codice include:
- implementazione MCMC (Metropolis–Hastings);
- simulazioni Monte Carlo;
- analisi empirica su rendimenti finanziari (DAX e S&P 500);
- produzione delle figure presentate in tesi.
